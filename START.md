# 🚀 快速开始指南

## 🔐 配置API密钥

### 1️⃣ 安装依赖（必需）

```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2️⃣ 配置API密钥

```bash
# 复制示例配置文件
cp .env.example .env

# 编辑 .env 文件，填入你的 DeepSeek API 密钥
# DEEPSEEK_API_KEY=your-api-key-here
```

### 3️⃣ 运行Agent

```bash
python3 main.py
```
