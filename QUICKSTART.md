# 快速入门指南

## 🚀 5分钟快速上手

### 第1步：确认Python环境

打开终端（Terminal），输入：

```bash
python --version
```

如果看到类似 `Python 3.8.x` 的输出，说明环境没问题。

### 第2步：进入项目目录

```bash
cd plant_cat_agent
```

### 第3步：运行程序

```bash
python plant_cat_agent.py
```

### 第4步：开始使用

程序启动后，你会看到欢迎界面，现在可以开始使用了！

## 🎓 理解代码结构

### 主要文件

1. **plant_cat_agent.py** - 核心程序
   - 包含 `PlantCatAgent` 类
   - 这是AI Agent的大脑

2. **data/plant_database.py** - 植物数据库
   - 包含12种植物的详细信息
   - 类似于AI的知识库

3. **tests/test_agent.py** - 测试文件
   - 验证程序是否正常工作
   - 学习如何编写测试

### 关键概念

1. **类（Class）** - 蓝图或模板
2. **对象（Object）** - 根据蓝图创建的实例
3. **方法（Method）** - 对象可以执行的动作
4. **属性（Attribute）** - 对象的属性或状态

## 🔧 修改和定制

### 添加新植物

打开 `data/plant_database.py`，找到 `PLANT_DATABASE` 字典，添加新植物：

```python
"玫瑰": {
    "name": "玫瑰",
    "scientific_name": "Rosa",
    "toxic_to_cats": False,
    "toxicity_level": "无毒",
    "symptoms": "可能被刺扎伤",
    "toxic_parts": "刺",
    "description": "玫瑰对猫无毒，但要注意刺"
},
```

保存文件，重新运行程序，就可以查询"玫瑰"了！

### 修改输出格式

打开 `plant_cat_agent.py`，找到 `_format_toxic_plant_response` 方法，修改输出格式：

```python
def _format_toxic_plant_response(self, plant_info):
    response = f"""
🚨 危险！{plant_info['name']}对猫有毒！
详细原因: {plant_info['symptoms']}
    """
    return response
```

## ❓ 遇到问题？

### 问题1：提示 "command not found: python"

**解决方法**：尝试使用 `python3` 代替 `python`

### 问题2：提示 "No module named xxx"

**解决方法**：
- 确保在正确的目录下运行
- 本项目不需要安装额外包

### 问题3：中文乱码

**解决方法**：
- 确保终端编码为UTF-8
- Windows用户可能需要设置：`chcp 65001`
